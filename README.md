# BookManagement
Spring Book Store repository for CRUD operation

All screenshots of the application output are stored in the `SpringBook-Images` folder:  
`BookManagement/src/main/resources/static/SpringBook-Images`.  
Open this folder and double-click the PNG files to view the screenshots of the running application.
